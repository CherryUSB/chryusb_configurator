# Cherry USB Configurator
> 用于生成 USB 描述符的图形化配置工具.


# Electron11 + Vue3.2 + Vite2 + TypeScript

<br><br>

## 构建

``` bash

# python 2.7
# electron 11.4.12
# nodejs 14.18.0
# vue 3.2.29
# vite ^2.7.13

# install dependencies~
yarn

# reinstall dependencies~
rm -rf node_modules
yarn

# serve with hot reload at localhost:9080
yarn dev

# build electron application for production
yarn build

```
